# Observateur-d-v-nements-Windows

Dans le fichier Mon-Filtre-de-Log.xml vous trouverez mon filtrage personnalisé (Il s'affiche en 1 seule ligne depuis le transfert sur Github alors que sur la VM il était lisible.. pour preuve la capture ci-dessous !)

Ce filtre permet de surveiller les événements liés au service DNS côté client comme serveur, avec niveaux de criticité :
- Critique
- Erreur
- Avertissement
- Information

Les éléments surveillés sont le démarrage et l'arrêt du server DNS, les erreurs de résolution de nom, les échecs de chargement de zone et les problèmes de réplication
